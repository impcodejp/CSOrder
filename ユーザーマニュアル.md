# CSOrder ユーザーマニュアル

**バージョン**: 1.1  
**対象システム**: CSOrder（CS受注予実管理システム）  
**最終更新**: 2026-04-30

---

## はじめに

CSOrder は CS メンバーの受注データを管理し、担当者別・グループ別の売上・粗利・予算達成率を確認するためのデスクトップアプリケーションです。

---

## 画面構成

アプリを起動すると左側にナビゲーションメニューが表示されます。

| メニュー | 機能 |
|---------|------|
| データ取込 | 受注データCSVをシステムに読み込む |
| 受注一覧 | 取り込み済みの受注データを確認・検索する |
| 帳票 | 担当別・グループ別の集計レポートを確認する |
| 社員管理 | 社員情報と月次予算を登録・編集する |

---

## 1. 社員管理

> **最初にこの画面で社員情報と月次予算を登録してください。**

### 1-1. 社員一覧の確認

画面を開くと登録済みの社員が一覧表示されます。

### 1-2. 社員を追加する

1. 右上の「＋ 行追加」ボタンをクリックします
2. 追加された空行に以下を入力します

| 項目 | 内容 | 例 |
|------|------|----|
| 社員番号 | 1以上の数値 | `12345` |
| 社員名 | 氏名 | `山田 太郎` |
| グループ | 所属グループ名 | `営業1部` |
| グレード | 等級・職位 | `A` |
| 各月予算 | 月ごとの予算（千円単位） | `1000` |

3. 入力後、「更新」ボタンをクリックして保存します

> **注意:** 「更新」を押さないと変更内容は保存されません。

### 1-3. 社員を削除する

削除したい行の「削除」ボタンをクリックします。  
行が即時削除されます。「更新」ボタンで保存してください。

### 1-4. CSVで一括登録する

社員データをExcel等で作成したCSVファイルから一括登録できます。

**手順:**
1. 「レイアウト出力」ボタンでCSVテンプレートをダウンロードします
2. テンプレートをExcelで開き、社員データを入力して保存します
3. 「CSV取込」ボタンでそのファイルを選択します
4. 画面に内容が展開されたことを確認します
5. 「更新」ボタンをクリックして保存します

**CSVのルール:**
- 文字コード: UTF-8（BOM付き）
- 月別予算の順番: 4月〜3月の順（会計年度順）
- 金額の単位: 千円

**CSVヘッダー:**
```
社員番号,社員名,グループ,グレード,4月予算,5月予算,6月予算,7月予算,8月予算,9月予算,10月予算,11月予算,12月予算,1月予算,2月予算,3月予算
```

### 1-5. 現在のデータをCSV出力する

「CSV出力」ボタンをクリックし、保存先を指定します。

## 2. データ取込

受注データのCSVファイル（Shift-JIS形式）をシステムに読み込みます。

### 2-1. CSVファイルを取り込む

1. 「CSVファイルを選択」ボタンをクリックします
2. ファイル選択ダイアログで受注データのCSVファイルを選択します
3. プレビューが表示されます。内容を確認してください

   | 確認項目 | 内容 |
   |---------|------|
   | 担当者名 | 正しい担当者名が表示されているか |
   | 契約日 | 正しい日付が表示されているか |
   | 金額 | 売上・粗利の値が正しいか |

4. 問題がなければ「取込開始」ボタンをクリックします
5. 「〇件取り込みました」と表示されれば完了です

### 2-2. エラーが表示された場合

バリデーションエラーがある場合はプレビューが表示されず、エラー内容が一覧で表示されます。

**よくあるエラーと対処法:**

| エラー内容 | 原因 | 対処 |
|-----------|------|------|
| 案件番号が数値ではない | 案件番号列に文字が入っている | CSVの該当行を修正する |
| 売上合計が数値ではない | カンマ付き数値や文字が入っている | 数値のみに修正する |
| 案件名が空です | 案件名が未入力 | 案件名を入力する |
| 契約日の形式が正しくない | YYYY/MM/DD 以外の形式 | 日付形式を統一する |

> **ヒント:** 契約日が空欄の行は自動的にスキップされます（エラーにはなりません）。

### 2-3. CSVファイルの仕様

| 列名 | 型 | 必須 | 説明 |
|------|----|------|------|
| 案件番号 | 数値 | ○ | 案件を一意に識別する番号 |
| 案件名 | 文字列 | ○ | 案件名称 |
| 顧客名 | 文字列 | ○ | 顧客企業名 |
| 売上合計 | 数値 | ○ | 受注金額（千円単位） |
| 粗利合計 | 数値 | ○ | 粗利額（千円単位） |
| 契約日 | YYYY/MM/DD | ○ | 契約締結日（空欄行はスキップ） |
| 案件主担当者ID | 数値 | ○ | 社員番号 |
| 案件主担当者名 | 文字列 | ○ | 担当者氏名 |

- 文字コード: **Shift-JIS**（自動変換されます）
- 同一案件番号のデータは上書きされます

## 3. 受注一覧

取り込み済みの受注データを一覧で確認できます。

### 3-1. 一覧を表示する

画面を開くと全受注データが表示されます。

### 3-2. 絞り込み検索

- **担当者名フィルター**: 担当者名の一部を入力すると絞り込まれます
- **顧客名フィルター**: 顧客名の一部を入力すると絞り込まれます

文字を入力すると自動的に再検索されます。

## 4. 帳票

担当者別・グループ別の受注集計を確認できます。

### 4-1. 集計期間を指定する

1. 「期間」の入力欄に開始月と終了月を入力します（暦月: 4〜3）
2. 「更新」ボタンをクリックします
3. 指定した期間の集計が表示されます

**入力例:**

| 確認したい期間 | from | to |
|--------------|------|-----|
| 4月単月 | 4 | 4 |
| 4月〜6月（第1四半期） | 4 | 6 |
| 4月〜3月（年間） | 4 | 3 |

### 4-2. 担当別タブ

社員ごとの集計を確認できます。

| 列 | 内容 |
|---|---|
| 名前 | 社員名 |
| GROUP | 所属グループ |
| GRADE | グレード |
| 件数 | 受注件数 |
| 売上 | 売上合計（千円） |
| 粗利 | 粗利合計（千円） |
| 予算 | 月次予算合計（千円） |
| 達成率 | 粗利 ÷ 予算 × 100（%） |

- 各グループの下に**小計行**が表示されます
- 最下部に**全体合計行**が表示されます
- 達成率が **100%以上**: 緑色表示
- 達成率が **80%未満**: 赤色表示

### 4-3. グループ集計タブ

グループ単位での集計を確認できます。

| 列 | 内容 |
|---|---|
| グループ | グループ名 |
| 人数 | グループの社員数 |
| 件数・売上・粗利・予算・達成率 | 担当別と同様 |

### 4-4. CSVで出力する

各タブの「CSV出力」ボタンをクリックすると、表示中の集計データをCSVファイルに保存できます。

- 担当別: `CS受注予実管理表_担当別_{期間}.csv`
- グループ別: `CS受注予実管理表_グループ別_{期間}.csv`

## よくある質問（FAQ）

**Q. 取り込んだデータを修正したい**  
A. 修正したCSVファイルを再取込してください。同一の案件番号のデータは自動的に上書きされます。

**Q. 社員を削除したら帳票に表示されなくなった**  
A. 帳票は社員マスタに登録されている社員を基準に集計します。削除した社員のデータは帳票に表示されません。受注データ自体はDBに残っています。

**Q. 予算が「—」で表示される**  
A. その社員の該当月の予算が0（未設定）の場合、達成率は「—」と表示されます。社員管理画面で予算を入力してください。

**Q. CSVの文字化けが発生する**  
A. 受注CSVはShift-JIS形式である必要があります。UTF-8で保存されたCSVは文字化けする場合があります。

## データの保存場所

システムのデータは以下の場所に保存されます。

```
C:\Users\{ユーザー名}\AppData\Roaming\com.csorder.app\csorder.db
```

> **注意:** このファイルを削除するとすべてのデータが失われます。定期的にバックアップしてください。
